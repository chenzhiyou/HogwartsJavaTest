REST-assured:
    简约的接口测试DSL
    支持xml/json的结构化解析
    支持xpath/jsonpath/gpath解析方式
    对Spring的支持比较全面
    符合契约编程思想
语法格式：
    given()：可以设置测试预设
        param():URL查询参数
    when(): 所要执行的操作
        get():GET请求
        post():POST请求
    then(): 可以解析结果、断言
        statusCode: 响应状态码断言
    请求参数：
        直接拼接在URL中
        GET请求：param(): 查询参数
        POST请求：
            queryParam(): 查询参数
            formParam(): 表单参数
断言：
    响应结果类型：
        状态码：statusCode()
        响应头: header()
        内容: body()
JSON简介：
    是javaScript Object Notation的缩写
    是一种轻量级的数据交换格式
    是理想的接口数据交换语言
    请求：
        构造JSON请求体：
            JSON字符串
            HashMap对象+Jackson库
JSON Schema：
    是使用JSON格式编写的
    可以用来定义校验JSON数据的结构
    可以用来校验JSON数据的一致性
    可以用来校验API接口请求和响应
使用代理:
    开启代理工具监听请求
    配置代理：
        局部：通过proxy()方法
        全局：定义proxy对象
Headers：
    Http Headers也叫做HTTP消息头
    允许客户端和服务器传递附加消息
    由名称、冒号、具体的值组成
Cookie使用场景：
    添加Cookie的两种方式
        通过header()方法
        通过cookie()方法
from表单：application/x-www-form-urlencoded
    应用场景：
        数据量不大
        数据层级不深的情况
        通常以键值对传递
    使用：
        formParams()方法
